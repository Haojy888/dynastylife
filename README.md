# 古代人生 Web

一个基于浏览器的古代人生模拟器网页版本，包含人生流年、亲友、营生、科举、家产、押镖、博坊、雅戏小游戏等玩法。

## 运行

这是静态网页项目，可以直接打开 `index.html`，也可以用任意静态服务器运行：

```bash
python -m http.server 5174
```

然后访问：

```text
http://127.0.0.1:5174/
```

## 主要文件

- `index.html`：页面入口
- `app.js`：游戏主逻辑
- `game-data.js`：事件、题库和资源数据
- `styles.css`：界面样式
- `assets/`：游戏图片资源
