# 古代人生 Web

一个纯前端的古风人生模拟游戏。从抓周开始,读书、科举、营生、姻缘、置业,直到走完一生。

**在线游玩:<https://www.dynastylife.online>**

## 玩法特色

- **人生流年**:561+ 事件随年龄推进,每个选择都影响属性与命运
- **流年画卷**:家事、求学、官场、本业、节庆、牢狱、江湖、天下、宗族、地方与签运事件会自动匹配专属场景插画，并适配桌面和移动端裁切
- **王朝风云**:国力、朝局、粮价、灾疫与边患逐年演化，并联动官场、家产、商旅和日常开销
- **跨年国史**:大河决口、宫闱易主、边关烽火三条三幕主线，职业专长会改变解决方式与最终评价
- **科举仕途**:书院温课、乡试会试殿试,一路考取功名；从县衙书吏到内阁大学士，每级官职都有对应专案与履历
- **女性人生线**:女子不参加科举；女学扩为女红、琴艺、诗书、礼仪、算账、岐黄六艺，并有专属选择剧情与女医技艺考校
- **营生百业**:27 种营生；21 个非官职职业各有三选一本业专案，并按专长属性检定、记录办案成绩；亦可押镖行商，风险与收益并存
- **亲友姻缘**:托媒相看、成婚纳妾，可与妻子或侧室夜话、同游、照料并共度良宵；父母疾病、兄弟分家与子女教养会形成跨年故事
- **故人记忆**:重要亲友拥有性格、志向、营生与自主年度行动，也会记住玩家的照料、借贷和共同经历
- **宗族经营**:族谱展示先人、尊亲、配偶、子女姻亲与孙辈；兄弟分家、子女成婚会形成支房，宗祠、义田、族学、义仓、公中钱与六种家规可跨代经营
- **人物关系网**:亲友不再只与主角单线互动，族人之间会自主提携、借贷、结盟、争执与和解，并通过合族议事和宗族纪事留下长期影响
- **家产置业**:买田置宅、市集淘物,也可能撞见黑市
- **雅戏风月**:五子棋、投壶、促织(斗蟋蟀)、博坊押数；成年男子可在瓦舍寻欢，歌舞从业女性仅可工作出入，其他女子可冒险女扮男装并承担名誉与婚嫁后果
- **诗会文斗**:书院/宴会选项制对句，胜涨学识名望，败则出丑
- **旅中奇遇**:车马与押镖解锁足迹图鉴
- **地域社会**:清平、云州、洛城、江陵、梁都、昆北、苏堤、青崖八地拥有独立声望、地方风俗、产业与年度事件
- **山河风物**:八地分别拥有高清风景主图与轻量缩略图，贯通九州声望版图、地方详情、车马选路和抵达过场
- **地方势力**:商会、军府、士林、船帮、丝坊、寺院等十六个势力可长期拜会并结盟；职业专长会影响地方办事与交涉
- **迁居置产**:当地声望达标后可举家迁居；每处家产登记所在州府，地方声望和盟友会直接影响产业年入，地域人脉可跨代继承
- **赛季名号**:促织府城大赛与博坊连胜；博坊连输可进黑名单
- **联姻策略**:媒人处细看家世、彩礼、性情、生育与娘家势力
- **秘密身份**:可选暗事（假功名、通匪等），年利与败露并存
- **天赋命格**:18 种命格 + 34 种天赋,开局随机组合

## 技术说明

纯静态站点,无需构建,零依赖运行:

- `index.html` — 页面入口
- `app.js` — 游戏主逻辑(原生 JS,无框架)
- `game-data.js` — 事件、题库与资源映射数据
- `styles.css` — 界面样式(WebGL 光影背景 + 卡片辉光交互)
- `assets/` — 游戏图片资源(WebP 优化)
- `tools/` — 本地维护脚本(资源扫描/压缩,不参与部署)

存档保存在浏览器 localStorage,无后端、无追踪。

## 本地运行

```bash
python -m http.server 5174
# 访问 http://127.0.0.1:5174/
```

## 开发检查

维护脚本位于 `tools/`。首次使用先安装依赖，之后可一键执行语法检查与浏览器主流程测试：

```bash
npm ci --prefix tools
npm run test:syntax --prefix tools
npm test --prefix tools
```

浏览器测试会验证移动端无横向溢出、新建人生、连续推进至 12 岁、多存档刷新恢复、桌面端渲染、JavaScript 错误与资源加载失败。推送和拉取请求也会通过 GitHub Actions 自动执行同一套检查。

## 部署

推送到 main 分支后由 Vercel 自动部署。缓存策略见 `vercel.json`(资源长缓存,HTML/JS 即时更新)。
